# Evde Yaşam Web Uygulaması

Bu proje, aynı evi paylaşan bireylerin harcama, görev ve ödeme takibini sağlayan web tabanlı bir sistemdir.

## Kullanılan Teknolojiler
- C#
- ASP.NET MVC
- MSSQL
- Entity Framework

## Proje Özellikleri
- Harcama takibi ve yönetimi
- Görev planlama sistemi
- Ödeme kontrolü
- Kullanıcı bazlı veri yönetimi

## Açıklama
Proje, MVC mimarisi kullanılarak geliştirilmiş olup ilişkisel veritabanı yapısı ile çalışmaktadır. Sistem, kullanıcıların günlük yaşam organizasyonunu kolaylaştırmayı amaçlamaktadır.

## Not
Proje geliştirme sürecinde oluşturulmuş olup, kaynak kodlar yerel ortamda bulunmaktadır.
